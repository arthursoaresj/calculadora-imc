##### 🧮 Calculadora de IMC em Python

Projeto desenvolvido em Python utilizando Tkinter para cálculo de IMC com interface gráfica.



##### 🚀 Funcionalidades

* ✅ Validação em tempo real dos campos
* ✅ Tratamento de erros
* ✅ Classificação automática do IMC
* ✅ Interface moderna e intuitiva
  

##### 📊 Classificações

* Magreza grave
* Magreza moderada
* Magreza leve
* Saudável
* Sobrepeso
* Obesidade grau 1, 2 e 3
  

##### 🛠 Tecnologias utilizadas

* Python 3
* Tkinter
  

##### ▶ Como executar

1. Clone o repositório: https://github.com/arthursoaresj/calculadora-imc
2. Abra a pasta: calculadora-imc
3. Execute: main.py
